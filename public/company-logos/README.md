# Company Logos

Drop your logo photos here. Prefer naming as `{companyId}.png` (e.g. `google.png`). Custom filenames are also supported via the component mapping.

**Examples:**
- google.png
- amazon.png
- microsoft.png
- meta.png
- apple.png
- netflix.png
- tcs.png
- infosys.png
- wipro.png
- flipkart.png
- swiggy.png
- zomato.png
- mckinsey.png
- bcg.png
- deloitte.png
- jpmorgan.png
- goldman.png
- openai.png

Company IDs match `lib/companies.ts`. If a logo is missing, the app falls back to Clearbit API, then initials.
